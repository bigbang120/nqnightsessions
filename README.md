# enq-night-sessions

An automated, quantitative trading script designed to execute algorithmic positions on Nasdaq-100 Futures (NQ) specifically during the low-noise Asian Session via the MetaTrader 5 Python API.

## Architecture
- **Language:** Python 3.10+
- **Platform:** MetaTrader 5 (DZero Execution Environment)
- **Session Focus:** 19:00 - 23:00 EST (Asian Open / Tokyo Positioning)
